Comandos de Linux

cal 07 2024

El comando cal sirve para extraer el calendario que queramos, por ejemplo cal 07 2024 muestra el calendario de julio del año 2024.

cal 2024

El comando cal sirve para extraer el calendario que queramos, por ejemplo cal 2024 muestra el calendario año 2024.

arch

El comando arch en sistemas Unix y Linux muestra la arquitectura de la máquina. Muestra si es de 32 o 64 bits, por ejemplo.

cat /proc/interrupts

El comando cat /proc/interrupts muestra una lista de interrupciones del sistema en tiempo real, incluyendo el número de interrupciones generadas por cada dispositivo o controlador de hardware.

lsusb -tv

El comando lsusb -tv muestra la jerarquía de dispositivos USB conectados al sistema, incluyendo información sobre los buses USB y los dispositivos conectados a ellos.

find / -user user1

El comando find / -user user1 busca en todo el sistema archivos que pertenecen al usuario "user1".

uname -m

El comando uname -m muestra la arquitectura de la máquina. Por ejemplo, si devuelve "x86_64", significa que estás utilizando una arquitectura de 64 bits.

cat /proc/meminfo

El comando cat /proc/meminfo muestra información detallada sobre la memoria del sistema, incluyendo el uso actual de la memoria, la memoria disponible, el tamaño de la memoria física y otros detalles relacionados con la memoria del sistema.

date

El comando date muestra la fecha y hora actuales del sistema.

find /home/user1 -name "*.bin"

El comando find /home/user1 -name "*.bin" busca archivos con extensión .bin dentro del directorio /home/user1 y sus subdirectorios.

uname -r

El comando uname -r muestra la versión del kernel del sistema operativo.

cat /proc/swaps

El comando cat /proc/swaps muestra información sobre el espacio de intercambio (swap) utilizado por el sistema, incluyendo el tamaño, el tipo y el nombre del archivo o dispositivo de intercambio.

find /usr/bin -type f -atime +100

El comando find /usr/bin -type f -atime +100 busca archivos dentro del directorio /usr/bin que hayan sido accedidos hace más de 100 días.

dmidecode -q

El comando dmidecode -q muestra información del Sistema de Administración de Datos de la Información del Dispositivo (DMI), que proporciona detalles sobre el hardware del sistema, como la BIOS, la placa base, la memoria y otros componentes.

cat /proc/version

El comando cat /proc/version muestra información sobre la versión del kernel del sistema operativo.

find /usr/bin -type f -mtime -10

El comando find /usr/bin -type f -mtime -10 busca archivos dentro del directorio /usr/bin que hayan sido modificados en los últimos 10 días.

hdparm -i /dev/hda

El comando hdparm -i /dev/hda muestra información detallada sobre el disco duro identificado como /dev/hda. Esto incluye el modelo, el tamaño del disco, la velocidad de transferencia y otra información relevante sobre el disco.

cat /proc/net/dev

El comando cat /proc/net/dev muestra estadísticas de red, incluyendo información sobre la cantidad de bytes recibidos y transmitidos en cada interfaz de red disponible en el sistema.

find / -name \*.apt -exec chmod 755 ‘{}’ \;

El comando find / -name \*.apt -exec chmod 755 ‘{}’ \; busca archivos con extensión .apt en todo el sistema y luego intenta cambiar los permisos de estos archivos a 755.

hdparm -tT /dev/sda

El comando hdparm -tT /dev/sda realiza pruebas de lectura del disco /dev/sda y muestra el tiempo que tarda en leer tanto datos en caché como datos sin caché. Esto proporciona información sobre el rendimiento de lectura del disco.

cat /proc/mounts

El comando cat /proc/mounts muestra una lista de los sistemas de archivos actualmente montados en el sistema, junto con información sobre sus puntos de montaje y opciones de montaje.

find

El comando find se utiliza para buscar archivos y directorios en un sistema de archivos.

locate

El comando locate se utiliza para buscar archivos en una base de datos preexistente del sistema.

cat /proc/cpuinfo

El comando cat /proc/cpuinfo muestra información detallada sobre el procesador(es) del sistema, incluyendo su modelo, velocidad, número de núcleos, caché, entre otros detalles.

lspci -tv

El comando lspci -tv muestra la topología de los dispositivos PCI del sistema en forma de un árbol. Proporciona una vista jerárquica de los dispositivos PCI conectados al sistema y sus relaciones.

find / -name file1

El comando "find / -name file1" busca un archivo llamado "file1" en todo el sistema de archivos, comenzando desde el directorio raíz (/). Esto buscará recursivamente en todos los directorios y subdirectorios del sistema de archivos en busca de un archivo con el nombre "file1".

whereis

El comando "whereis" se utiliza para buscar la ubicación de archivos binarios, fuentes y páginas de manual relacionadas con un comando específico en el sistema.

kill -9 ID_Processo

El comando kill -9 ID_Processo se utiliza para enviar una señal SIGKILL al proceso con el ID_Processo especificado, lo que causa la terminación inmediata del proceso.

kill -1 ID_Processo

El comando kill -1 ID_Processo se utiliza para enviar una señal SIGHUP (hang-up) al proceso con el ID_Processo especificado.

lsof -p $$

El comando lsof -p $$ muestra todos los archivos abiertos por el proceso actual (representado por $$). Esto incluye archivos de datos, bibliotecas compartidas, sockets de red, entre otros recursos que están siendo utilizados por el proceso en el momento en que se ejecuta el comando.

lsof /home/user1

El comando lsof /home/user1 muestra todos los archivos abiertos por procesos que están ubicados dentro del directorio /home/user1.

tail

El comando tail se utiliza para mostrar las últimas líneas de un archivo.

who

El comando who muestra información sobre los usuarios que están actualmente conectados al sistema, incluyendo su nombre de usuario, terminal de inicio de sesión, fecha y hora de inicio de sesión y dirección IP desde la que se conectaron, si está disponible.

alias

El comando alias se utiliza para crear, ver y eliminar alias de comandos en la terminal. Los alias son atajos o nombres alternativos que puedes asignar a comandos largos o complejos para facilitar su uso.

wget

wget es un comando utilizado para descargar archivos desde la web a través de HTTP, HTTPS y FTP.

whois

El comando whois se utiliza para obtener información sobre un dominio de Internet, como el propietario del dominio, la dirección de correo electrónico del contacto administrativo, el servidor de nombres y otra información relacionada.

nslookup

El comando nslookup se utiliza para consultar servidores de nombres de dominio (DNS) para obtener información sobre la resolución de nombres de dominio.

host

El comando host también se utiliza para realizar consultas DNS y obtener información sobre la resolución de nombres de dominio.

tar

El comando tar se utiliza para crear, manipular o extraer archivos y directorios comprimidos en formato tar.

tar -cvf archive.tar file1

El comando tar -cvf archive.tar file1 crea un archivo tar llamado "archive.tar" que contiene el archivo "file1". El significado de las opciones utilizadas:

- -c: Crea un nuevo archivo tar.
- -v: Muestra el progreso detallado de la operación.
- -f: Especifica el nombre del archivo tar que se creará.

En este caso, se crea un archivo tar llamado "archive.tar" que contiene el archivo "file1".

tar -cvf archive.tar file1 file2 dir1

El comando tar -cvf archive.tar file1 file2 dir1 crea un archivo tar llamado "archive.tar" que contiene los archivos "file1" y "file2", así como el directorio "dir1" y todos sus contenidos.

El significado de las opciones utilizadas:

- -c: Crea un nuevo archivo tar.
- -v: Muestra el progreso detallado de la operación.
- -f: Especifica el nombre del archivo tar que se creará.

Después de -f, debes especificar el nombre del archivo tar que se creará, y luego la lista de archivos y/o directorios que deseas incluir en el archivo tar.

tar -tf archive.tar

El comando tar -tf archive.tar muestra el contenido del archivo tar "archive.tar" sin extraerlo.

- -t: Muestra el contenido del archivo tar.
- -f: Especifica el nombre del archivo tar del cual se desea ver el contenido.

Esto mostrará una lista de todos los archivos y directorios incluidos en "archive.tar".

tar -xvf archive.tar

El comando tar -xvf archive.tar extrae el contenido del archivo tar "archive.tar" en el directorio actual.

- -x: Extrae el contenido del archivo tar.
- -v: Muestra el progreso detallado de la operación.
- -f: Especifica el nombre del archivo tar del cual se desea extraer el contenido.

Después de -f, debes especificar el nombre del archivo tar del cual se extraerá el contenido.

tar -xvf archive.tar -C

El comando tar -xvf archive.tar -C intenta extraer el contenido del archivo tar "archive.tar" en el directorio especificado después de la opción -C.

tar -cvfj archive.tar.bz2 dir1

El comando tar -cvfj archive.tar.bz2 dir1 crea un archivo tar comprimido en formato bzip2 llamado "archive.tar.bz2" que contiene el contenido del directorio "dir1".

- -c: Crea un nuevo archivo tar.
- -v: Muestra el progreso detallado de la operación.
- -f: Especifica el nombre del archivo tar que se creará.
- -j: Utiliza la compresión bzip2.

Después de -f, debes especificar el nombre del archivo tar que se creará, y luego la lista de archivos y/o directorios que deseas incluir en el archivo tar. En este caso, solo se incluye el directorio "dir1".

tar -xvfj archive.tar.bz2

El comando tar -xvfj archive.tar.bz2 extrae el contenido del archivo tar comprimido en formato bzip2 "archive.tar.bz2" en el directorio actual.

- -x: Extrae el contenido del archivo tar.
- -v: Muestra el progreso detallado de la operación.
- -f: Especifica el nombre del archivo tar del cual se desea extraer el contenido.
- -j: Utiliza la compresión bzip2.

Esto extraerá el contenido del archivo tar comprimido "archive.tar.bz2" en el directorio actual.

tar -cvfz archive.tar.gz dir1

El comando tar -cvfz archive.tar.gz dir1 esto creará un archivo tar comprimido en formato gzip llamado "archive.tar.gz" que contiene el contenido del directorio "dir1".

tar -xvfz archive.tar.gz

El comando tar -xvfz archive.tar.gz esto extraerá el contenido del archivo tar comprimido "archive.tar.gz" en el directorio actual.

shutdown -h now

El comando shutdown -h now se utiliza para apagar el sistema de inmediato. La opción -h indica que el sistema se apagará después del cierre, mientras que now especifica que el apagado debe ocurrir inmediatamente.

init 0

El comando init 0 se utiliza para apagar el sistema. En sistemas basados en SysV init, el nivel de ejecución 0 está asociado con el apagado del sistema. Sin embargo, en algunos sistemas modernos que utilizan systemd, init 0 puede no tener el mismo efecto que en sistemas SysV init, ya que systemd maneja el apagado de manera diferente.

telinit 0

El comando telinit 0 es otra forma de apagar el sistema en sistemas basados en SysV init. En estos sistemas, telinit es una interfaz para cambiar el nivel de ejecución del sistema. El nivel de ejecución 0 está asociado con el apagado del sistema.

halt

El comando "halt" se utiliza para detener el sistema de forma inmediata. Detiene todos los procesos y apaga el sistema. Es similar a "shutdown -h now" pero es más directo y no realiza ninguna otra acción antes de apagar el sistema.

shutdown -h hours:minutes &

El comando shutdown -h hours:minutes & se utiliza para programar un apagado del sistema en un momento específico, especificado por las horas y los minutos. La opción -h indica que el sistema se apagará después del cierre.

shutdown -c

El comando shutdown -c se utiliza para cancelar un apagado o reinicio programado del sistema que haya sido previamente configurado con el comando shutdown. Al ejecutar shutdown -c, se cancelará cualquier apagado o reinicio programado que esté en espera.

shutdown -r now

El comando shutdown -r now se utiliza para reiniciar el sistema de inmediato. La opción -r indica que el sistema se reiniciará después del cierre, mientras que now especifica que el reinicio debe ocurrir inmediatamente.

reboot

El comando reboot se utiliza para reiniciar el sistema de forma inmediata. Al igual que shutdown -r now, reboot reinicia el sistema de inmediato sin realizar ninguna otra acción adicional.

logout

El comando logout se utiliza para cerrar la sesión actual de usuario en la terminal. Al ejecutar logout, se cerrará la sesión actual y se volverá al inicio de sesión o al prompt de inicio de sesión, dependiendo del entorno de terminal utilizado.

exit

El comando exit se utiliza para cerrar la sesión actual de usuario en la terminal o para salir del shell actual. Al ejecutar exit, se cerrará la sesión actual y se volverá al inicio de sesión o al prompt de inicio de sesión, dependiendo del entorno de terminal utilizado.

vi

El comando "vi" se utiliza para abrir el editor de texto Vi en la terminal. Vi es un editor de texto modal que proporciona potentes funciones de edición y es ampliamente utilizado en sistemas Unix y Linux.

nano

El comando "nano" se utiliza para abrir el editor de texto Nano en la terminal. Nano es un editor de texto simple y fácil de usar que es popular entre los usuarios que prefieren una interfaz más amigable que la de Vi o Vim.

cat

El comando "cat" se utiliza para mostrar el contenido de uno o varios archivos en la terminal.
